DOM Manipulation When Scrolled To Certain Elements

Small example page of running an animation, or effectively anything you would like to do, once the user has scrolled to a certain element on the page. Works completely dynamically, letting it work on any screen size, from desktop to mobile

Color Guessing Game

A small RGB guessing game. Initial idea from the Udemy course "The Web Developer Bootcamp" by Colt Steele. While taking the course, I took the code-along project and instead created it on my own based on the specifications Colt gave and the way his example looked. I took things a step further, playing around with CSS transitions, CSS animations, as well as animations with javascript, to make the application feel a little more tactile and interesting.
